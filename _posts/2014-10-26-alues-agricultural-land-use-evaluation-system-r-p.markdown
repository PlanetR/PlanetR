Asaad</b> (maintainer)<br /><i>alstated@gmail.com</i><br /><br />Agricultural Land Use Evaluation System (ALUES) is an R package that evaluates land suitability for different crop production. The package is based on the Food and Agriculture Organization (FAO) and the International Rice Research Institute (IRRI) methodology for land evaluation. Development of ALUES is inspired by similar tool for land evaluation, Land Use Suitability Evaluation Tool (LUSET). The package uses fuzzy logic approach to evaluate land suitability of a particular area based on inputs such as rainfall, temperature, topography, and soil properties. The membership functions used for fuzzy modeling are the following: Triangular, Trapezoidal and Gaussian. The methods for computing the overall suitability of a particular area are also included, and these are the Minimum, Maximum, Product, Sum, Average, Exponential and Gamma. Finally, ALUES uses the power of Rcpp library for efficient computation.<br /><br /><h3>INSTALLATION</h3>The package is not yet on CRAN, and is currently under development on github. To install it, run the following:<br /><br /><script src="https://gist.github.com/alstat/383a072fc51b4f6e526d.js"></script>We want to hear some feedbacks, and if you have any suggestion or issues regarding this package, please do submit it <a href="https://github.com/alstat/ALUES/issues" target="_blank">here</a>.<a name='more'></a><br /><br /><h3 style="text-align: left;">DATASET</h3>The package contains several datasets which can be categorized into two: <br /><ol style="text-align: left;"><li><b>Land Units' Attributes</b> - datasets that contain the attributes of the land units of a given location.</li><li><b>Crop Requirements</b> - datasets that contain the required values of factors of a particular crop for the land units.</li></ol><h3>Land Units' Attributes</h3>The package contains sample dataset of land units' attributes from two countries:<br /><ol style="text-align: left;"><li><b>Marinduque, Philippines</b>: <ul><li> <code>MarinduqueLT</code> - a dataset consisting the water characteristics of the land units of Marinduque, Philippines.</li></ul></li><li><b>Lao Cai, Vietnam</b><ul><li><code>LaoCaiLT</code> - a dataset consisting the land and terrain characteristics of the land units of Lao Cai, Vietnam;</li><li><code>LaoCaiTemp</code> - a dataset consisting the temperature characteristics of the land units in Lao Cai, Vietnam;</li><li><code>LaoCaiWater</code> - a dataset consisting the water characteristics of the land units of Lao Cai, Vietnam;</li></ul></li></ol>For example, the first six land units in <code>MarinduqueLT</code> is shown below<br /><br /><script src="https://gist.github.com/alstat/d43faeaec0ed1475c809.js"></script>The complete list of factors is available in the pdf version. <br /><br /><h3>Crop Requirements</h3>The crops available in the package are the listed in Table 1. <br /><br /><div class="datagrid"><table><thead><tr><th>Code</th><th>Crops</th></tr></thead><tfoot><tr><td colspan="4" style="text-align: center;"><div id="paging"><i>Table 1: Crops Dataset Available in ALUES.</i></div></td></tr></tfoot><tbody><tr class="alt"><td><code>BANANA-</code></td><td>Banana</td></tr><tr><td><code>CASSAVA-</code></td><td>Cassava</td></tr><tr class="alt"><td><code>COCOA-</code></td><td>Cocoa</td></tr><tr><td><code>COCONUT-</code></td><td>Coconut</td></tr><tr class="alt"><td><code>COFFEEAR-</code></td><td>Arabica Coffee</td></tr><tr><td><code>COFFEERO-</code></td><td>Robusta Coffee</td></tr><tr class="alt"><td><code>RICEBR-</code></td><td>Rainfed Bunded Rice</td></tr><tr><td><code>RICEIW-</code></td><td>Irrigated Rice</td></tr><tr class="alt"><td><code>RICENF-</code></td><td>Rice Cultivation Under Natural Floods</td></tr><tr><td><code>RICEUR-</code></td><td>Rainfed Upland Rice</td></tr></tbody></table></div><br />From the table, the codes are suffixed with the land units' characteristics (<code>TerrainCR</code>, <code>SoilCR</code>, <code>WaterCR</code> and <code>TemperatureCR</code>) required for the crop. Other fuzzy models are <code>"Trapezoidal"</code> and <code>"Gaussian"</code>.</td></tr><tr><td style="text-align: right;" valign="top"><code>sow.month</code></td><td>sowing month of the crop. Takes integers from 1 to 12 (inclusive), representing the twelve months of a year. So if sets to 1, the function assumes sowing month on January.</td></tr><tr><td style="text-align: right;" valign="top"><code>min</code></td><td>factor's minimum value. If <code>NULL</code> (default), <code>min</code> is set to 0. But if numeric of length one, say 0.5, then minimum is set to 0.5 for all factors. If factors on land units (<code>x</code>) have different minimum, then these can be concatenated to vector of <code>min</code>s, the length of this vector should be equal to the number of factors in <code>x</code>. However, if sets to <code>"average"</code>, then <code>min</code> is theoretically computed as:<br /><br />Let X be a factor, then X has the following suitability class: S3, S2 and S1. Assuming the scores of the said suitability class for X are $a, b$ and $c$, respectively. Then, $$\mathrm{min} = a - \displaystyle\frac{(b - a) + (c - b)}{2}$$ For factors with suitability class S3, S2, S1, S1, S2 and S3 with scores $a, b, c, d, e$ and $f$, respectively. <code>min</code> is computed as, $$\mathrm{min} = a - \displaystyle\frac{(b - a) + (c - b) + (d - c) + (e - d) + (f - e)}{5}$$</td></tr><tr><td style="text-align: right;" valign="top"><code>max</code></td><td>factor's maximum value. Default is set to <code>"average"</code>. If numeric of length one, say 50, then maximum is set to 50 for all factors. If factors on land units (<code>x</code>) have different maximum, then these can be concatenated to vector of <code>max</code>s, the length of this vector should be equal to the number of factors in <code>x</code>. However, if sets to <code>"average"</code>, then <code>max</code> is computed from the equation below: $$\mathrm{max}=c + \displaystyle\frac{(b-a) + (c-b)}{2}$$ For factors with suitability class S3, S2, S1, S1, S2 and S3 with scores $a, b, c, d, e$ and $f$, respectively. Then, $$\mathrm{max} = f + \displaystyle\frac{(b - a) + (c - b) + (d - c) + (e - d) + (f - e)}{5}$$</td></tr><tr><td style="text-align: right;" valign="top"><code>interval</code></td><td>domain for every suitability class (S1, S2, S3, and N). If <code>"fixed"</code>, the interval would be 0 to 0.25 for N (Not Suitable), 0.25 to 0.50 for S3 (Marginally Suitable), 0.50 to 0.75 for S2 (Moderately Suitable), and 0.75 to 1 for (Highly Suitable). If <code>"unbias"</code>, then the interval is set to 0 to $\displaystyle\frac{a}{\mathrm{max}}$ for N, $\displaystyle\frac{a}{\mathrm{max}}$ to $\displaystyle\frac{b}{\mathrm{max}}$ for S3, $\displaystyle\frac{b}{\mathrm{max}}$ to $\displaystyle\frac{c}{\mathrm{max}}$ for S2, and $\displaystyle\frac{c}{\mathrm{max}}$ to $\displaystyle\frac{\mathrm{max}}{\mathrm{max}}$ for S1.</td></tr></tbody></table></center><br /><b>Output</b><br />The function returns the following output:<br /><ol><li>Actual Factors Evaluated;</li><li>Suitability Score;</li><li>Suitability Class;</li><li>Factors' Minimum Values; and,</li><li>Factors' Maximum Values.</li></ol><i>Example</i>: To test the suitability of the land units in Marinduque, Philippines, for terrain requirements of coconut, we have<br /><br /><script src="https://gist.github.com/alstat/dac48c584bc347a35924.js"></script>Before we run the function, let's check for the possible output.
