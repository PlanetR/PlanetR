<strong>Package</strong>: cusp<br>
<strong>Type</strong>: Package<br>
<strong>Title</strong>: Cusp-Catastrophe Model Fitting Using Maximum Likelihood<br>
<strong>Version</strong>: 2.3.1<br>
<strong>Author</strong>: Raoul P. P. P. Grasman [aut, cre, cph]<br>
<strong>Maintainer</strong>: Raoul Grasman &lt;rgrasman@uva.nl&gt;<br>
<strong>LazyData</strong>: yes<br>
<strong>NeedsCompilation</strong>: yes<br>
<strong>Suggests</strong>: plot3D<br>
<strong>Description</strong>: Cobb's maximum likelihood method for cusp-catastrophe modeling
(Grasman, van der Maas, &amp; Wagenmakers, 2009, JSS, 32:8;
Cobb, L, 1981, Behavioral Science, 26:1, 75--78).
Includes a cusp() function for model fitting, and several
utility functions for plotting, and for comparing the
model to linear regression and logistic curve models.<br>
